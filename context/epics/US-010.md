# [US-010] Cover images para albums
## Descripcion
Los 4 albums no tienen cover_image. La pantalla de albums muestra placeholder gris.
## Problema
Sin cover image los albums se ven vacios y la home screen no tiene carrusel atractivo.
## Criterios de aceptacion
- Cada album tiene un cover_image subido
- El cover se muestra en la lista de albums, home screen y profile screen
