# [US-005] Sticker detail con datos reales
## Descripcion
El sticker detail muestra specs hardcodeadas (velocidad, HP, etc.). Reemplazar con datos reales del backend o eliminar las specs falsas.
## Problema
Las specs del carro (293 km/h, 4.2s, 385 HP, etc.) estan hardcodeadas. El boton "Anadir al album" solo muestra un SnackBar sin llamar API.
## Criterios de aceptacion
- Mostrar descripcion real del sticker en la card de "dato curioso"
- Mostrar puntos, rareza, album y fecha de captura (si aplica) con datos del backend
- Quitar specs falsas o alimentarlas desde un campo del modelo
- El boton principal debe tener funcionalidad real o no existir
