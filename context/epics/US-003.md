# [US-003] Mensaje personalizado en sticker desbloqueado
## Descripcion
Permitir que el usuario escriba un mensaje/comentario en un sticker que ya desbloqueo, visible en el sticker detail.
## Problema
`ContentApi.setStickerMessage()` existe pero ningun screen lo usa. El campo `message` del sticker no se muestra.
## Criterios de aceptacion
- En sticker detail, si el sticker esta desbloqueado, mostrar campo de mensaje
- Poder escribir y guardar un mensaje
- El mensaje se persiste en backend via POST /stickers/{id}/message/
