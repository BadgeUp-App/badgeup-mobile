# [US-002] Editar sticker existente
## Descripcion
Pantalla para editar nombre, descripcion, rareza, puntos e imagen de referencia de un sticker ya creado.
## Problema
`ContentApi.updateSticker()` existe pero no hay pantalla de edicion. Un admin no puede corregir datos de un sticker.
## Criterios de aceptacion
- Desde sticker detail, boton de editar (solo visible para admins)
- Formulario pre-llenado con datos actuales
- Poder cambiar imagen de referencia
- Al guardar, regresa al detail actualizado
